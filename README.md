# the-last-shinobi
# Casual 3D Runner

The project is a casual 3D runner created as part of
a test task for the position of Unity game developer.

## Requirements

- Unity version 2022.3.15f1
- Zenject

## Controls

- Use the A and D keys to move left and right, respectively.
